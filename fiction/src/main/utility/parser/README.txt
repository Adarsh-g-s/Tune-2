EXECUTION MECHANISM OF JSON PARSER

1. Ensure that the zips are placed in a location. (Here named zipFileLocation)
2. Zip names are given.
3. open json_parser_config and then add the properties that were created earlier.
4. You can give 3 options for outputs:
	1. csv
	2. txt
	3. prop
5. Defaulted option is prop only!!!

6. Now, enjoy running the parser script and feel free modify anything
